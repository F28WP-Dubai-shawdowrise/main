About our Game



Shadow Rise is a 1V1 Shooter Multiplayer game in which players are assigned to 2 teams, Cop and Criminal. It is a horizontal base with multiple platforms lying all over the map. The cops and criminals have guns which have a long reload time and a health bar. The objective of the game is to kill each other. The first one to reduce the other one's health wins the game.

This was the initial version of the game with just blocks moving around, with no score,health bar and no map design.




This is the final version of the game with complete game mechanics working and a redesigned map with multiple platforms making the game challenging.
